# ai-sessions
Contains hands on tutorials for AI learnings.

## Pre-Requisite
- Install Podman
- Install Python (IfNotPresent) locally ~ From v3.11+ (Only required if running locally)
- System with sufficient space ~ Around 3 GB (To download and run models)

## Usage

- Clone the repo
```bash
git clone https://github.com/mayuka-c/ai-sessions.git
```

- Follow the README of individual directories to learn more.
